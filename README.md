# django-tutorial

Django Tutorial project from steps 1 to 4 with the list of all polls in the index page.

Create virtual environment and activate it:
```
virtualenv -p python3 env
source env/bin/activate
```

Developed using `Django 3.2.5`, to install this last realease, inside your virtualenv type:
```
pip install Django
```

To run the project, inside `mysite` folder and with the previous steps completed, type:
```
python manage.py runserver
```
